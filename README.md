# Engineering Project Template

> A lightweight, modular and AI-friendly standard for starting software projects.

---

## Overview

Engineering Project Template provides a universal project structure designed for collaboration between humans and AI agents.

It focuses on:

* simplicity by default;
* progressive documentation;
* minimal cognitive load;
* single source of truth;
* long-term maintainability.

The template is technology-agnostic and complements the Engineering Playbook.

The design principles behind these choices, and the relationship with the Engineering Playbook, are described in [philosophy.md](philosophy.md).
